The editor's main surfaces:

- **Canvas** — the central DAG view, where you drag nodes and wire connections
- **Node palette** — left sidebar with the built-in node kinds and a "New Script" button per language
- **Inspector** — right panel; shows fields for the selected node, plus a "Preview" tab with its current output
- **Script editor** — Monaco-backed pane that opens when you select a `script` node
- **Run console** — bottom panel with per-node run progress and captured stdout
