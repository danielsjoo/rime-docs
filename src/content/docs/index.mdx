## Why Rime

<CardGrid stagger>
  <Card title="One DAG, four languages" icon="puzzle">
    SQL for joins, Python for ML, R for stats, JavaScript for everything else. Same pipeline, named slots, typed boundaries.
  </Card>
  <Card title="Reproducible by default" icon="seti:lock">
    Content-addressed caching, deterministic outputs, freeze-able snapshots. No "works on my machine."
  </Card>
  <Card title="Publishable narratives" icon="document">
    Pair your DAG with a `report.yaml` and render a publishable HTML report. Tables, stats, prose, all in one render step.
  </Card>
  <Card title="Visual editor (preview)" icon="laptop">
    Rime Editor is a desktop app for authoring pipelines visually. Closed beta; download coming soon.
  </Card>
</CardGrid>
