Rime ships with curated example pipelines that double as documentation. Each one is a complete project — DAG, scripts, data, and a rendered HTML report.

## Flagship

- **[Cars × CO₂ emissions](/examples/cars-emissions/)** — SQL source + JS API fetch + Python UMAP + R regression → one HTML narrative. The canonical multi-language example.

## Teaching

- **[Penguin classifier](/examples/penguins/)** — Smallest interesting DAG. Pivot, derive, t-test, plot. Best place to start after the [Quick start](/get-started/quick-start/).
- **[DAG showcase](/examples/dag-showcase/)** — Walk through every core node kind with a tiny example each.
- **[Single-file pipeline](/examples/single-file/)** — A DAG that lives in one YAML file with no project marker. The smallest possible Rime project.

## Integration

- **[Embed in Node](/examples/headless/)** — Use `@rimekit/runtime` programmatically from any Node script. For CI plugins and custom dashboards.
- **[DuckDB single source](/examples/sql-only/)** — Pure-SQL pipeline using DuckDB ingress mode. No Python or R required.
